Usage: 
---------- 
1) Download and unzip the responsive-login-page.zip file.
2) Five files named "index.html, README.txt, jquery.js, script.js and style.css " with images folder will be obtained. 
3) Run the index.html file in your web browser to see the functionality of the script. 

Index.html: 
----------
This file contains the entire functionality of "signup and login". 

Images Folder: 
------------------ 
1) Images folder contains the images required for the script. 
2) Replace the images in images folder with your own images. 

jquery.js: 
-----------
jquery.js file contains the jquery library functions that supports the jquery animations. Do not change the code in this file. If any changes are done in this file, then it will affect the entire functionality of the script.
 
script.js:
--------- 
1. script.js file contains the positions of the forms of the script and their animating effects.
2. The script will work only when the script.js file is included.

How to embed the script in to your webpage? 
-------------------------------------------------------------- 
1) Copy the content inside the "body" from index.html file and paste the code inside the <body></body> of your webpage where you want to display the effect. 
2) You can replace the images and css properties according to your needs.
3) Make sure that the script.js, style.css and jquery.js files and image folder are placed in appropriate folder. Include these files in <head> of your page.
